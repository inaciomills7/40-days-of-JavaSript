🚀 Day 04 - 40 Days of JavaScript

This repository contains my solutions for Day 04 of the 40 Days of JavaScript challenge from tapaScript.

The focus of this day is to improve problem-solving skills using:

- `switch` statements
- Conditional logic
- Real-world scenarios

📌 Tasks Completed

✅ 1. Switch Case Behavior

Understood how `switch` works and why it is **case-sensitive**.

---

✅ 2. ATM Cash Withdrawal System

Built a system to:

- Allow withdrawals only in multiples of 100
- Validate user input before processing

---

✅ 3. Calculator (Switch Case)

Implemented a basic calculator using `switch`:

- Addition (+)
- Subtraction (-)
- Multiplication (\*)
- Division (/)
- Modulus (%)

---

✅ 4. Movie Ticket Pricing

Calculated ticket prices based on age:

- 👶 Children (<18): $3
- 🧑 Adults (18–60): $10
- 👴 Seniors (60+): $8

✅ 5. Horoscope Sign Checker

Used `switch` (without `if-else`) to determine zodiac signs based on "month input".

> ⚠️ Note: This is a simplified version (month-based, not date-based).

✅ 6. Triangle Type Checker

Determined triangle type based on side lengths:

- Equilateral
- Isosceles
- Scalene

Also added validation to ensure the sides form a "valid triangle".

- `switch` vs `if-else`
