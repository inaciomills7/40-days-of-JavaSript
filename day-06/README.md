🚀 Day 06 - 40 Days of JavaScript

This repository contains my solutions for Day 06 of the 40 Days of JavaScript challenge from _tapaScript_.

The focus of this day is mastering functions, improving logical thinking, and understanding execution flow in JavaScript.

---

📌 Tasks Completed

✅ 1. Celsius to Fahrenheit Converter

Created a function to convert temperature from Celsius to Fahrenheit using the formula:

(C × 9/5) + 32

---

✅ 2. Find Maximum of Two Numbers

Built a function to return the larger of two numbers using comparison operators.

---

✅ 3. Palindrome Checker

Created a function to check whether a string reads the same forward and backward.

---

✅ 4. Factorial Calculator

Implemented a function to calculate the factorial of a number using a loop.

Example:

5! = 5 × 4 × 3 × 2 × 1 = 120

---

✅ 5. Count Vowels in a String

Developed a function to count vowels (`a, e, i, o, u`) in a given string.

---

✅ 6. Capitalize Words in a Sentence

Wrote a function to capitalize the first letter of each word in a sentence.

---

✅ 7. IIFE (Immediately Invoked Function Expression)

Used an IIFE to print a message to the console with parameters.

---

✅ 8. Callback Function

Created a function that accepts another function as a parameter and executes it.

---

✅ 9. Call Stack Execution (Flow 1)

Analyzed how functions are pushed to and popped from the call stack.

---

✅ 10. Call Stack Execution (Flow 2)

Understood execution order when multiple functions are invoked.

---

🧠 Key Concepts Learned

- Function declaration and usage
- Parameters and return values
- String manipulation using loops
- Callback functions

- Call stack and execution flow
- Writing reusable and modular code
